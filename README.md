### MolClustPy: A Python package to analyze multivalent biomolecular clusters 

Please visit [MolClustPy website](https://molclustpy.github.io/index) for documentation and tutorials.

The "molclustpy_package" folder contains data for the PyPI repository, which provides the convenient installation via "pip install molclustpy". pyBionetgen should automatically be installed with molclustpy.
